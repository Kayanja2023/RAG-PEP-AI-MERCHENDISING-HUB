# Hollard Policy Assistant Knowledge Base

This directory contains the knowledge base documents for the Hollard Policy Assistant.

## Required Documents

To fully enable the assistant, add the following markdown files to this directory:

1. **hollard-products-overview.md** - Insurance products (Life, Disability, Business, etc.)
2. **hollard-faqs.md** - Frequently asked questions
3. **life-insurance-basics.md** - Complete guide to life insurance
4. **claims-process.md** - Step-by-step claim procedures
5. **about-hollard.md** - Company information and Better Futures mission

## Document Format

All documents should be in Markdown (.md) format and contain comprehensive information about Hollard Insurance products and services.

You can also upload documents through the Streamlit app interface.
